# Machine Learning / AI Engineering Bootcamp — Learning Portfolio

A structured, day-by-day portfolio of Python data analysis and machine learning exercises, built as part of a self-directed bootcamp on the path toward becoming an ML/AI engineer. Each script focuses on a specific concept and is framed around a realistic business or institutional scenario (school management, retail sales, healthcare risk) rather than abstract textbook data.

This repository is intended to show progression: starting with core NumPy/Pandas data handling, moving into classification and scoring logic, and ending with a multi-domain capstone that combines several of the earlier concepts.

## Contents

| Day | File | Focus |
|---|---|---|
| 1 | `student_marks_analyzer_day1.py` | NumPy fundamentals — array creation, filtering, descriptive statistics on student marks data |
| 2 | `student_performance_classifier_day2.py` | Conditional logic and classification rules to bucket students by performance level |
| 3 | `sales_analytics_system_day3.py` | Sales data aggregation — totals, averages, min/max, identifying high vs low performing periods |
| 4 | `student_attendance_+_performance_Tracker_day4.py` | Combining attendance and performance metrics into a single tracking view |
| 5 | `health_risk_analyzer_day5.py` | Threshold-based health risk flagging using age and related indicators |
| 6 | `mini_data_dashboard_system_day6.py` | Bringing multiple metrics together into a simple console-based dashboard |
| 7 | `CAPSTONE_AI_multi_domain_Data_analyzer_day7.py` | Capstone — combines student, sales, and health analysis logic from earlier days into a single multi-domain script |

## Tech Stack

Python, NumPy, Pandas — focused on array operations, filtering, aggregation, and basic rule-based analysis.

## Context

This bootcamp directly informs **SmartBiz Gambia AI**, a planned business intelligence application (Flask/MySQL) featuring health risk prediction, loan readiness scoring, and revenue forecasting. The exercises here are deliberately scoped around those same domains so each concept has a clear path to a real application feature.

The same structured handouts produced during this bootcamp (covering NumPy and Pandas) are also used as teaching material for software development students at St. Stephen Institute of Technology.

## Status

Early-stage learning portfolio — scripts are intentionally simple and console-based at this point, prioritizing correct use of core data analysis concepts before moving into scikit-learn models, visualization, and the SmartBiz application itself.
